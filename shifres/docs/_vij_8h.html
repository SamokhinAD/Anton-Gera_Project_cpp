Подробное описание

Шифр Виженера - простой и надежный способ шифрования текста

Этот код позволяет зашифровать или расшифровать любой текст с помощью ключевого слова. Принцип работы: каждая буква текста сдвигается на позицию соответствующей буквы ключа

Главная функция для работы с шифром Виженера

Аргументы
alphabetНабор символов, которые можно шифровать (обычно буквы алфавита)
key_textКлючевое слово для шифрования (будет повторяться по кругу)
textТекст, который нужно зашифровать или расшифровать
modРежим работы: 0 - выбор с клавиатуры, 1 - шифрование, 2 - расшифрование
Возвращает
Результат работы - зашифрованный или расшифрованный текст

Шифр работает по принципу: каждая буква исходного текста складывается с соответствующей буквой ключа (для шифрования) или вычитается (для расшифрования)
