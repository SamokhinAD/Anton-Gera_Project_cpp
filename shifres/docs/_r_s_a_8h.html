Подробное описание

RSA шифрование - самый популярный алгоритм с открытым ключом

Этот код реализует базовую версию RSA для шифрования чисел. RSA работает на основе сложности факторизации больших чисел - легко перемножить два числа, но сложно разложить результат обратно

Аргументы

Результат операции - зашифрованное или расшифрованное число

RSA использует пару ключей: публичный (n,e) для шифрования и приватный (n,d) для расшифрования. Безопасность основана на том, что сложно найти d, зная только n и e
