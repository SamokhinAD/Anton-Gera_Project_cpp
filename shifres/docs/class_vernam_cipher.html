Подробное описание

Реализация шифра Вернама (одноразовый блокнот)

Класс предоставляет функциональность для шифрования и дешифрования с использованием алгоритма Вернама (XOR-шифрование). Примечание: Для реальной криптографической стойкости ключ должен быть истинно случайным и равным по длине сообщению (одноразовый блокнот).
