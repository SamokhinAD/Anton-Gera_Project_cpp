Главная программа для работы с различными шифрами 

Это основная программа, которая позволяет выбрать один из трёх шифров\+: аффинный, Виженера или RSA. После выбора запускается соответствующий алгоритм шифрования. Управление происходит через нажатие клавиш

Показывает меню выбора шифров и ждёт нажатия клавиш\+:
\begin{DoxyItemize}
\item 1\+: запускает аффинный шифр
\item 2\+: запускает шифр Виженера ~\newline

\item 3\+: запускает RSA
\item Esc\+: выход из программы 
\end{DoxyItemize}Флаг для отслеживания первого нажатия клавиши (чтобы не реагировать на повторные)

Код нажатой клавиши

Выход из программы при нажатии Esc (код 27)

Запуск аффинного шифра при нажатии \textquotesingle{}1\textquotesingle{} (код 49)

Запуск шифра Виженера при нажатии \textquotesingle{}2\textquotesingle{} (код 50)

Запуск RSA при нажатии \textquotesingle{}3\textquotesingle{} (код 51)
